command: map-folder
title: Map the folder
description: Understand the files, datasets, schemas, and likely data pipeline.
hidden: true
---

Map the current workspace before proposing changes.

Focus on:
- Important files and directories.
- Datasets, schemas, time-series sources, and generated artifacts.
- How data likely flows from raw inputs through transforms, analysis, storage, and outputs.
- Gaps where the repository structure does not make the pipeline clear.

Return a compact map of what exists, what each part appears to do, and what should be inspected next.
